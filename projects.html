Capstone: SatiLLM

Description

An exploratory project from Argonne National Laboratory's Applied Security Head, focusing on the possibility of using LLMs for identifying anomalous space conditions. The project existed in two haves, one focusing on ns-3 satellite simulation and the other on LLM integration.

How I Contributed

I headed the AI side of the project, going in with no prior LLM deployment experience. I begain devising a setup that used cutting edge models and size minification techniques. I eventually designed and implemented a decentralized RAG-equipped Agentic AI using a custom backend adapter, using the then-new ChatGPT-OSS 20B as our model. By doing this, I was able to leverage existing resources to successfully design the system using none of our project budget, with the setup being lightweight enough to run off my laptop and an old workstation card by effectively distributing the load placed on system resources.

Skills and Knowledge Gained

Agentic AI [dspy], RAG [ColBERT v2], Multi-System Orchestration, AI Backend Development [HyperLocal], LLMs

Real-Life RE2 Electronic Lockbox Puzzle [Digital Logic Class Project]

Description

Having recently gone through the classic game's remaster, I was inspired to create a physical version of it's Lockbox puzzle. By using an FPGA, I was able to bring this puzzle into reality, using a similar LED indicator mechanism as well as the ability to alter the solution without needing to reprogram the board, by using a series of button inputs with an 7-segment display.

Skills and Knowledge Gained

Rapid prototyping, FSM design, FPGA Programming

Resources Used

Intel Quartus Prime, Altera DE2-115 Development Board [Cyclone IV FPGA], Falstad Circuit Sim, Game References

G24 SEASHELL

Description

A fully featured educational wargame-style project as my digital forensics project. It is designed to teach students about identifying inconsistencies and discovering data exfiltration channels, revolving around the same technology NASA uses to communicate with Voyager. Although it used to require a supercomputer to do this kind of math, a laptop GPU is now capable of the calculations.

Skills and Knowledge Gained

Resources Used

NVidia CUDA Toolkit, Numba, Python, NVIDIA GeForce mobile RTX card, VSCode
