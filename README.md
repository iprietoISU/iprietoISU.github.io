# iprietoISU.github.io

This site uses GitHub Pages and Jekyll Sass.

- Edit theme and layout styles in `assets/css/styles.scss`.
- GitHub Pages compiles `assets/css/styles.scss` into `assets/css/styles.css` during deployment.
- The checked-in `assets/css/styles.css` is a local preview fallback and is excluded from the GitHub Pages build in `_config.yml`.
